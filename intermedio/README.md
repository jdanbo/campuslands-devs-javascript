# Nivel intermedio - JavaScript

Este nivel contiene 80 ejercicios progresivos para practicar JavaScript con problemas mas cercanos a situaciones reales.

## Como esta organizado

Los ejercicios suben de dificultad de forma gradual:

- 01-20: funciones, arrays, objetos, filtros, rankings y estadisticas.
- 21-40: reglas de negocio, simulaciones, scoring, clases basicas y datos relacionados.
- 41-60: DOM, eventos, formularios, fechas, localStorage y componentes simples.
- 61-80: mini apps e integraciones con estado, persistencia, reportes y proyectos mas completos.

## Regla de entrega

Cada estudiante debe entregar dentro de la carpeta `resoluciones` del ejercicio correspondiente:

```text
intermedio/ejercicios/01-nombre-del-ejercicio/resoluciones/nombre-apellido/
```

No modifiques `README.md` ni `ejercicio.js` base.
